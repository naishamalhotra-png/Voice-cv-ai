# 🎙️ Voice CV AI

Build professional resumes using just your voice.

Voice CV AI is an AI-powered resume builder that converts spoken conversations into structured, ATS-friendly resumes. The application leverages Gemini for intelligent content generation and Sarvam AI for speech processing, enabling users to create high-quality resumes through a simple voice-based experience.

## ✨ Features

* 🎤 Voice-based resume creation
* 🤖 AI-powered content generation with Gemini
* 🗣️ Speech-to-text processing using Sarvam AI
* 📄 Professional and ATS-friendly resume output
* ⚡ Fast, intuitive, and user-friendly interface

## 🛠️ Tech Stack

* Frontend: React / Next.js
* AI Model: Gemini
* Speech Processing: Sarvam AI
* Deployment: Vercel

## 🚀 How It Works

1. Record your professional details through voice.
2. Sarvam AI converts speech into text.
3. Gemini structures and enhances the content.
4. Generate a polished resume in seconds.
